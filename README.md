## chezmoi.dots

---

### fresh start

```bash
chezmoi init --apply brazcom
```

---

### daily usage

write local changes to chezmoi:

```bash
chezadd.sh
```

commit and push:

```bash
chezpush.sh
```

---

[chezadd.sh](http://chezadd.sh) monitored directories:

```bash
  "$HOME/.config/scripts"
  "$HOME/.config/rofi"
  "$HOME/.config/i3"
  "$HOME/.config/installers"
  "$HOME/.config/kitty"
  "$HOME/.config/polybar"
  "$HOME/.config/alacritty"
  "$HOME/.local/share/applications"
```
